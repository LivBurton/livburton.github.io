Example Projects

MusicSight

A mobile-first responsive music website where you can search for artists and lyrics. Using a music API with async await, vanilla JavaScript and CSS.

Movie App

An app to search for movies and create a watch list using Fetch API, vanilla JavaScript using classes, and Bootstrap with additional CSS. Local Storage is used so that the watch list persists.

Currency Converter

A currency coverter app using Fetch API, fetching data from JSON files, vanilla JavaScript and CSS. The user has the choice whether to search by country, for further information, or just by currency code.

To-Do List App

Using HTML, JavaScript and Bootstrap. The app allows the user to add and delete individual tasks, cross them off when completed and delete all tasks in the list. I have added task information to local storage, so that the data persists, and the ability to filter the task list.

An Example Recipes Website

Pages using CSS, HTML and JavaScript. Here JavaScript is used to change the navigation menu for smaller screens and scroll through and filter different recipes.
