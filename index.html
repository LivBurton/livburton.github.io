About Me

Hi, I'm Liv!

I'm a self-taught Front-End Web Developer from Manchester, UK, with skills in HTML, CSS, JavaScript, Bootstrap and responsive web design. I'm really passionate about blending creativity with coding, overcoming technical challenges and learning new technologies; I'm currently learning and putting React into practise.

My first experience with coding was during my PhD, where I self-learned C++ to produce object-oriented simulation models and published in several journals. I am now working hard to transition from being a secondary school Teacher and Head of Year into an exciting career in web development!

I'm also very passionate about self-defence; I'm a qualified instructor and teach in my spare time.

Example Projects

MusicSight

A mobile-first responsive music website where you can search for artists and lyrics. Using a music API with async/await, JavaScript and CSS.

Movie App

A responsive app to search for movies and create a watch list using Fetch API, vanilla JavaScript with classes, and Bootstrap with additional CSS. Local Storage is used so that the watch list persists.

Currency Converter

A currency coverter app using Fetch API, fetching data from JSON files, vanilla JavaScript and CSS with responsive design. The user has the choice whether to search by country, for further information, or just by currency code.

To-Do List App

Using HTML, JavaScript and Bootstrap. The app allows the user to add and delete individual tasks, cross them off when completed and delete all tasks in the list. I have added task information to local storage, so that the data persists, and the ability to filter the task list.
